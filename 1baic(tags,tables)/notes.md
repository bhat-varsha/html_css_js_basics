Tag Selector (Element Selector) :A tag selector applies styles to all elements of that tag.
h1{} p{ this is tag selectort }
When you want to style every element of the same type.
_____________________________________________________
class selector is used when you want to style multiple specific elements, not all of them.
Classes start with a dot . in CSS.
You can apply the same style to many different elements.
_____________________________________________________
ID Selector : An ID selector is used for one unique element.
IDs start with # in CSS.
An ID should be used only once in a page.
_____________________________________________________
Use Tag selector : when styling all elements

Use Class selector :  when styling many specific elements

Use ID selector : when styling one unique element

Use Div : to group elements into section
